# B10315031_HW3
RSA加解密

##  建置環境
**OS:** &nbsp;macOS 10.12.5   
**Programming Language:** &nbsp;Python 3.6.1

##  完成功能
1.	RSA加解密

2.	可變金鑰長度(bit)
	*	1024
	*	2048
	*	4096
	
3.  加速
	*	Square & Multiply
	*	Chinese remainder theorem

4.	質數
	*	找質數方法(Miller-Rabin Test)	
	*	產生大質數

5.  GUI
